MeshCollection

Events expected:

  • 'onMeshLoaded': whenever a mesh is loaded. the callback of this event is called with the arg:
new MeshCollection(threeContext: any, mesh: THREE.Mesh, id: string, id: string, step: string, progress: number, error: Error, id: string, message: string, id: string)

Extends EventManager

Parameters
threeContext (any = null)
mesh (THREE.Mesh) mesh object
id (string) id of the mesh (as used within this collection)
  • 'onMeshLoadingProgress': when the loading status is updated. The callback arguments are:
id (string) id of the element that could not be loaded
step (string) name of the step being in progression (ie. 'parsing')
progress (number) percentage of progress on the 'step'
  • 'onMeshLoadError': whenever a mesh could not be loaded, for various reasons. Args of the callbac:
error (Error) the error explaining what was wrong
id (string) id of the element that could not be loaded
  • 'onMeshLoadWarning': when mesh is asked to be loaded but it's already being processed or in the loaded. Note that this is based on the ID, not the URL
message (string) the explanation
id (string) ID of the mesh being loaded

▸ loadMeshFromUrl(url, options)

Load a mesh file from a distant file, with the provided url.

loadMeshFromUrl(url: string, options: object)
Parameters
url (string) the url to load the file from
options (object = {}) the options object
Name Description
options.format string must be one of: 'obj' (no others for the moment :D )
options.id string the id to attribute to the mesh once it will be part of the collection. Automatically generated if not provided
options.makeVisible boolean if true, the mesh will be added and made visible once loaded. If false, it's just going to be parsed and will have to be added later using its id (default: true)
options.color string the color to apply to the mesh in the format '#FFFFFF' (default: '#FFFFFF', does not apply if a material is given)
options.focusOn boolean once loaded, the camera will look at it
options.material THREE.Material the material to apply to this mesh (default: a generated Fresnel material)
